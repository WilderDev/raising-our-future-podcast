# Raising Our Future

Podcast player and feed for Raising Our Future Podcast. Building a strong foundation for tomorrow's champions

## Features

- Audio Player w/ Controls
- Mobile Responsive
- RSS Feed
- Simple, clean design


## Tech Stack

**Client:** NextJS, TailwindCSS, Prisma

**Server:** NodeJS, Vercel
